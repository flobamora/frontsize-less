Main features

  • Migration capable

    You can totally custom your grid rules and override other frameworks to unleash
    the extreme Frontsize flexibility.

  • CSS lint test proof

    At the moment the only one front-end framework which passes CSS lint tests, to
    let you do the best quality work.

  • Fallbacks

    Do you need fallback compatibility? You can enable Frontsize features for IE,
    activate / deactivate browser prefixes, or use fallback compatibility mixins.
